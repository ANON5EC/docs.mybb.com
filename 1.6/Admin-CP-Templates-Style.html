The Templates & Style tab in the Admin CP contains everything that has to do with the layout and design of your forum.

Themes are the style and design part of your board, using CSS. You can have several different themes on your forum. Each theme can have multiple stylesheets, and you can edit each style sheet in either a basic or advanced mode. Read the full document »

Templates are the layout part of your board, using HTML. You can create different template sets to be used by your theme, which each contain all of the template groups, which contain the templates themselves. Template editing uses Codepress. Read the full document »
