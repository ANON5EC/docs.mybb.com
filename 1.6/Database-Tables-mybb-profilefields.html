Stores custom profile fields.

Field Type Null Key Default Description
fid smallint(5) No Primary Field ID
name varchar(100) No Field name
description text No Field description
disporder smallint(5) No 0 Display order
type text No Type of field
length smallint(5) No 0 Length of field
maxlength smallint(5) No 0 Maximum length of field
required int(1) No 0 '1'/'0' requiring users to fill out this field
editable int(1) No 0 '1'/'0' whether or not users can edit this field in their UCP
hidden int(1) No 0 '1'/'0' whether or not this field is hidden in the Front End
