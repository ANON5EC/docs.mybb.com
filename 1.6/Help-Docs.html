Help Documentation

Several Help Documents come with MyBB by default. Others can be added through the Admin CP. The default help documents are written to help new users understand the basics of MyBB.

The Help Documentation is accessed by clicking on "Help" at the top of the screen. It provides a listing of all help documents, split into their respective categories. Each category can be collapsed and expanded. To view a help document, simply click on its name.
